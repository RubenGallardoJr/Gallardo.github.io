PacMan Exercise: This code is for a game where Pac-Man characters move randomly on the screen and bounce back when they hit the edges. The code creates a Pac-Man with a random position and velocity and adds it to a div element in the HTML. The movement of the Pac-Men is animated by changing their position based on their velocity and making them bounce back when they reach the edges.

Learn more about this game at PacMan Exercise Repository.

Eye Exercise a webpage will be created that includes two eyes that follow the movement of your mouse. This is done by utilizing mouse events such as "mousemove" to bind the document object model elements to events.

Learn more about this game at Eye Exercise Repository.

Real Time Bus Tracker ExerciseThe Real-Time-Bus-Tracking exercise consists of using an API to source data from the Boston MBTA busses in real-time. This data is then used to display the multiple stops between MIT and Harvard on a map.

Learn more about this game at Real Time Bus Tracker Exercise Repository.
